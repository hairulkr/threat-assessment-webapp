  • CVSS base score analysis with temporal scoring
  • Exploit availability and patch status assessment
  • Attack complexity and user interaction requirements

Exploit Ranking Agent

  • Public exploit availability and weaponization status
  • Exploit reliability and success rate analysis
  • Attack vector complexity and prerequisites

Authority Ranking Agent

  • Source credibility and verification status
  • Official vs community source weighting
  • Historical accuracy and false positive rates

Relevance Ranking Agent

  • Product and technology stack matching
